# MacawsHytaleMods
Macaw's Hy Mods is a collection of the Macaw's Mod series for Hytale, this includes the following mods:

    Macaw's Hy Furniture
    Macaw's Hy Windows
    Macaw's Hy Doors
    Macaw's Hy Paintings
    Macaw's Hy Light & Lamps
    Macaw's Hy Paths & Pavings
    Macaw's Hy Carpets
    Macaw's Hy Stairs
    Not Enough Poops
